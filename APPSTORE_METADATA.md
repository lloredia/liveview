## Description
Track live sports scores, match timelines, and real-time updates across soccer, basketball, hockey, baseball, and football — all in one beautiful app.

LiveView gives you instant access to scores from 24+ leagues including the Premier League, La Liga, Serie A, Bundesliga, NBA, NFL, MLB, NHL, MLS, Champions League, and more.

KEY FEATURES:

• Live Scores — Real-time score updates with ESPN data integration
• 24+ Leagues — Soccer, basketball, football, hockey, and baseball
• Match Detail — Play-by-play timelines, player stats, team stats, and lineups
• League Standings — Full standings tables updated in real-time
• Head-to-Head — Historical matchup data between teams
• Team Form — Recent W/D/L results at a glance
• Score Alerts — Push notifications for goals and key events
• Track Matches — Pin up to 3 matches for quick monitoring
• Live Activities — See scores on your Lock Screen and Dynamic Island
• News Feed — Latest sports news from ESPN, BBC Sport, Sky Sports, and more
• Dark & Light Mode — Beautiful themes that match your system preference
• Favorites — Follow your teams and leagues for a personalized experience

Whether you're a soccer fan following the Premier League, a basketball enthusiast tracking the NBA playoffs, or a baseball fan keeping up with MLB scores — LiveView has you covered.

## App Review Notes
This app displays live sports scores and news. It requires an internet connection to fetch score data. The app uses a server-side API for match data and ESPN's public API for supplementary live scores.

Test the app by:
1. Opening the app — the Today view shows all matches for the current date
2. Tapping any match card to see match details
3. Using the sidebar (hamburger menu) to browse leagues
4. Switching between dark and light mode via the theme toggle in the header
5. Tapping "News" to see the latest sports articles

No login is required to use the app.
